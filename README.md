Nombre del proyecto: Vectores de Numeros positivos y negativos



Descripcion: Este programa realiza con un llenado automatico de vectores, mostrando desde vectores positivos y negativos de forma aleatoria.

Luego se imprimen en 2 maneras, de forma horizontal y vertical. Por ultimo se contabilizan los números mayores



Lenguaje: Java 



Funciones: contarPositivos: Recorre el vector y devuelve la cantidad total de números mayores a cero.

contarNegativos: Analiza el vector para identificar y contar cuántos números tienen un valor menor a cero.

contarMayores: Recibe un valor entero como parámetro y retorna cuántos elementos del vector superan dicho límite.

LlenadoAutomatico:Llena el vector con números aleatorios dentro del rango especificado por el usuario.



Como ejecutar: Desde la terminal, compila y ejecuta la clase principal:

javac Mavenproject7.java VectorEntero.java
